# Minimal Image Search color options

这组不重新发明图标，只围绕当前应用图标的配色做更鲜明的变化。

## 推荐

- `option-01.svg`：最适合直接迭代当前图标，棕色更深、太阳更亮，整体仍然稳。
- `option-02.svg`：更活泼，橙色存在感更强，适合希望工具栏图标更醒目。
- `option-05.svg`：加入少量青绿色边线，和暖棕形成冷暖对比，辨识度最好但风格变化更明显。

## 候选

- `option-01.svg`：明亮琥珀，当前图标的增强版。
- `option-02.svg`：橙色冲击，颜色更鲜明。
- `option-03.svg`：金色高对比，偏稳定高级。
- `option-04.svg`：珊瑚暖色，更年轻。
- `option-05.svg`：青绿色边线，冷暖对比最明显。
- `option-06.svg`：紫色边线，最跳，但可能和当前 UI 关系较远。

选定编号后，再导出正式 `icon-16.png`、`icon-32.png`、`icon-48.png`、`icon-128.png`。
