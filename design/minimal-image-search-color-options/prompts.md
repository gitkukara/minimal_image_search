# Color exploration notes

Goal: keep the current Minimal Image Search icon composition, then improve color recognition.

Base motif:

- Magnifying glass
- Image frame
- Mountain shape
- Sun accent

Palette direction:

- Keep walnut / espresso brown as the main structural color.
- Make the sun more vivid.
- Use warmer ivory backgrounds with higher contrast.
- Test one or two accent-ring colors only where useful.

Constraints:

- Do not replace production icons before selection.
- Keep all source files square.
- Keep the silhouette readable at 16px and 32px.
- Avoid text, letters, numbers, or unrelated symbols.
