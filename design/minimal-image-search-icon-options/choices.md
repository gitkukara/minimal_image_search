# Minimal Image Search icon options

Generated for the Chrome / Edge image-search extension. These are candidate source icons only; none of them has replaced the production icon yet.

## Shortlist

- `option-01.svg`：最接近当前识别资产，适合做稳妥迭代。
- `option-04.svg`：把截图选择能力放进图标语义里，功能感最强。
- `option-06.svg`：32px 下轮廓最粗，浏览器工具栏可读性最好。

## Options

- `option-01.svg`：放大镜包裹图片框，延续当前品牌方向。
- `option-02.svg`：图片卡片叠在放大镜下，层次更轻。
- `option-03.svg`：眼睛和搜索结合，偏“视觉识别”。
- `option-04.svg`：扫描角标加放大镜，偏截图搜索。
- `option-05.svg`：图片堆叠加搜索，强调多图/素材感。
- `option-06.svg`：最大胆的风景轮廓，适合小尺寸。
