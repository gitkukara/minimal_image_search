# Icon generation notes

Product: Minimal Image Search

Use case: Chrome / Edge extension icon for reverse image search.

Visual direction: minimal warm UI, image search, magnifying glass, screenshot/image recognition, readable at 32px, no letters or words inside source icons.

Palette:

- Warm off-white: `#f7f4ed`
- Panel ivory: `#fffaf0`
- Walnut brown: `#4b2f19`
- Secondary brown: `#7a5434`
- Amber accent: `#f8bf3c`

Constraints:

- Square source files.
- Opaque background.
- Centered bold symbol.
- No text, letters, numbers, watermarks, app screenshots, or existing brand marks.
- Do not overwrite production icons before selection.
